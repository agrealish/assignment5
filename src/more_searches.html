More Searching Methods

Now that we have introduced you to the simple text search, we can get into a little detail on ways you can enhance your text search, and other ways to filter your data.

Text Searches with Keywords

There are keywords that can be used in the simple text search shown on the prior page. This will allow you to search particular fields for matching text, instead of searching every field as the samples on the prior page does. The following table details some of the keywords you can use in your text query searches:

Keywords to use in Text Searches
Keyword What it Does
intitle This searches the title for the text following this keyword.
inauthor This searches the author for the text following this keyword.
subject This searches the subject or category for the text following the keyword.
inpublisher This searches the publisher for the text following this keyword.
isbn This searches the ISBN Number for the text following this keyword.

There are other fields that you can use to search; the above is just a subset of the fields available to search in the text search.

The formatting of this search is much the same as the text search of the prior page, just with the addition of some keywords. For instance, if we wanted to search for The Great Hunt by Robert Jordan, we could use the following keyword search:

https://www.googleapis.com/books/v1/volumes?q=hunt+inauthor:jordan&key=yourAPIKey

Notice that the 'inauthor' keyword was used in the above example, and it was used as part of the 'q' string and connected with a '+'.

Here is the example code from the prior page with using the above search

  <?php
  include 'storedInfo.php';
  
  //use curl to send query to the Google Books API
  $searchurl = "https://www.googleapis.com/books/v1/volumes?q=hunt+inauthor:jordan&key=" . $myAPIKey;
  $curl = curl_init();
  curl_setopt($curl, CURLOPT_URL, $searchurl);
  curl_setopt($curl, CURLOPT_FOLLOWLOCATION,1);
  curl_exec($curl);
  curl_close($curl);
  ?>

Parameters Other Than 'q' can be Used in Searches

As the title of this section indicates, parameters other than the 'q' we have been using can be used in searches. If used in conjunction with the 'q' parameter, they must be concatenated on the search string using the &.

Filter

One of these other parameters is the filter parameter. This can be used to obtain only certain results. For instance, setting the filter parameter equal to free-ebooks returns only those results that are free Google eBooks.

Filter can also be set equal to values such as paid-ebooks, ebooks, etc.

Example getting only the free ebooks of Robert Jordan's books:

https://www.googleapis.com/books/v1/volumes?q=robert+jordan&filter=free-ebook&key=yourAPIKey

Print Type

Another parameter that can be used to get specific results is the printType parameter. This can be used to restrict the results of your query to those that are just books, or just magazines. An example code is included below.

https://www.googleapis.com/books/v1/volumes?q=robert+jordan&printType=books&key=yourAPIKey
