Other Types of Searches

We are briefly going to cover other types of searches other than the text search previously covered. Specifically, we will briefly cover how to search by Volume ID, and how to search bookshelves.

Searching For A Volume

One way that you can search on the Google Books API, is by searching for a specific volume. A volume ID is a unique string that is given to each volume that Google Books knows about, so you do need to have prior knowledge of the Volume ID in order to search this way. Your text search does return the volume ID, so subsequent searches for the same book can be made easire by this search method.

Example of using ID to search:

https://www.googleapis.com/books/v1/volumes/yngEsxEO4QYC?key=yourAPIKey

Searching a Bookshelf

Bookshelves are ways that specific users who utilize the Google Books app as part of their Google tools can save books to their account. As you may have guessed, some of this data is public and some of this data is more private. With the API Key, we will only be able to see bookshelves that have been made public.

One thing with searching specific bookshelves is that you need to have the ID of the user whose bookshelves you want to view in order to search them. So if you don't have the user ID, you cannot see that bookshelf.

To see a user's public bookshelves, you can use the following code format:

https://www.googleapis.com/books/v1/users/[userId]/bookshelves

Again, to use this code, you need the user ID of the user whose bookshelves you want to see. For this reason, I am not going to include specific examples for this part as I would rather not point to a specific user in an example.

To actually see the books on a specific bookshelf, you can use the following code format:

https://www.googleapis.com/books/v1/users/[userId]/bookshelves/[shelf]/volumes?key=yourAPIKey

Note the addition of the input [shelf] (and recall that this needs to be a public bookshelf for us to retrieve anything). The value [shelf] represents a numeric value that corresponds to a particular shelf for a user. Google provides some pre-defined shelves for every user, which are detailed below:

Shelf IDs
Shelf ID Title of Shelf
0 Favorites
1 Purchased
2 To Read
3 Reading Now
4 Have Read
5 Reviewed
6 Recently Viewed
7 My eBooks
8 Books For You - if Google does not have recommendations, then this shelf does not exist
